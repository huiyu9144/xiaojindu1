📖 Progress - Support

About the App

Progress is a minimalist project progress management app that helps you easily track multiple projects' completion status.

How to Use

  1. Create a New Project
    Click the "+" button in the top right corner to create a new project
  2. Edit Name
    Tap the project name to edit directly
  3. Adjust Progress
    Drag the slider on the progress bar to adjust the progress percentage
  4. Delete Project
    Swipe left on a project to show the delete button
  5. Reorder Projects
    Long press a project name and drag to reorder projects

Data Storage

All app data is stored locally on your device (UserDefaults). There is no cloud synchronization feature.
